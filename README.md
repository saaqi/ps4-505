# PS4 5.05 Lightweight Exploit Host

This is a small, lightweight exploit host for PlayStation 4 systems running firmware **5.05**.

I built this mainly for my own use as a clean, no-frills alternative to larger or more feature-heavy hosts. It’s designed to be easy to run locally, quick to load in the PS4 browser, and simple to maintain.

## Features

- Minimal and lightweight setup
- Fast-loading exploit pages
- Local hosting (no external dependencies required)
- Easy to modify or extend for personal needs

## Purpose

This repository exists primarily as a personal archive and reference.
It’s not intended to be a polished, user-friendly project or a general-purpose solution for others.

## Usage

- Host the files using a local web server of your choice
- Access the host through the PS4 browser on firmware 5.05
- Follow your usual exploit workflow

Exact setup details may vary depending on your environment.

## Notes

- This project targets **PS4 firmware 5.05 only**
- No support is provided
- Use at your own risk
